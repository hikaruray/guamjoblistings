# GuamJobListings 公開手順書（オーナー用）

公開作業はすべて無料です。あなたの作業は**3つのアカウント連携だけ**で、合計30分ほど。各ステップで私（CEOエージェント）が設定値を用意し、貼り付け箇所まで案内します。

---

## 全体像

```
① Vercel    … サイトを世界に公開する場所（無料）
② Resend    … 応募・投稿メールを送る鍵（無料・月3,000通）
③ ドメイン接続 … www.guamjoblisting.com を ① に繋ぐ
④ データベース … 応募・投稿を本番保存（Supabase 無料）
```

---

## ① Vercel（サイト公開）

1. https://vercel.com に GitHub または メールで無料登録
2. このプロジェクトを Vercel に取り込む（私が手順を案内）
3. 自動で `〇〇.vercel.app` の仮URLが発行され、まず動作確認できる

> あなたの作業：登録のみ。デプロイ設定は私が行います。

## ② Resend（メール送信の鍵）

1. https://resend.com に無料登録
2. APIキーを1つ発行（コピーするだけ）
3. そのキーを Vercel の環境変数 `RESEND_API_KEY` に貼り付け（私が場所を案内）

> これで応募が「企業へ直送＋あなたにBCC控え」で実際に飛びます。

## ③ ドメイン接続

1. ドメインを買った管理画面（お名前.com 等）を開く
2. 私が用意するDNSレコード（数行）を貼り付け
3. 数十分〜数時間で www.guamjoblisting.com が本番サイトに

## ④ データベース（Supabase）

1. https://supabase.com に無料登録
2. プロジェクトを1つ作成 → 接続情報をコピー
3. Vercel の環境変数に貼り付け（私が用意するテーブル作成SQLを実行）

> これで応募・投稿が本番でも消えずに Admin に溜まります。

---

## 公開後すぐやること（私が下準備）

- お付き合いのある **3〜5社の求人を掲載**（内容をいただければ私が反映）
- Google Search Console に登録（無料・SEOの土台、私が手順案内）
- 各社へ「掲載しました」のお知らせ（英語テンプレを私が用意）

---

## あなたが用意するもの

- 各サービスのログイン（メールアドレス）
- 実際の求人情報（会社名／職種／時給／勤務地／仕事内容／応募の届け先メール）

準備ができたら「公開しよう」と言ってください。順番に進めます。
