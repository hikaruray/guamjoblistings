## 0. 確定方針（オーナー決定 2026-07-02）

以下はオーナーが決定した確定方針。第3〜4章の「案1」を採用し、価格・無料期間を具体化したもの。
本章が今後の運用の基準。第1〜5章は検討の経緯として残す。

### 採用モデル
- **案1 フリーミアム軸**を採用。段階移行: **フェーズ0 全掲載無料で母数づくり → フェーズ1 アドオン課金 → フェーズ2 将来サブスク**。

### 無料期間
- **「掲載数が育つまで無料」**（一律◯ヶ月ではなく、母数が一定に達するまで全掲載無料）。
- **到達基準の目安案【推測・要検証】**（いずれかを満たしたらアドオン課金＝フェーズ1へ移行を検討）:
  - 実在企業の掲載が **20〜30社** に到達、または
  - アクティブ求人が **50件** を超える、または
  - 月間の応募数が **継続して発生**（例: 月20件以上が2ヶ月連続）。
  - ↑数値はグアム市場規模（民間5万人規模）を踏まえた初期目安。実データを見て調整する。

### アドオン価格（フェーズ1で導入）
- **安め $10〜20/件** から検証（低賃金・高回転市場に合わせ「気軽に試せる」水準）。当初の$25–40案は不採用。
  - 例: Featured（上位表示） **$15–20 / 30日** / 急募バッジ **$10前後** / 掲載延長・少額パック も同水準帯。
- 反応を見て段階的に調整。

### 残り論点の確定（秘書くん判断）
- **決済手段**: 導入時は **Stripe** を第一候補（他プロジェクトと統一）。実装は課金フェーズ着手時。
- **最初に有料化を狙う業種**: **ホテル・飲食・ウォータースポーツ**（反復採用組）。
- **競合の実額調査**: 優先度低・当面不要（母数づくり優先）。

### 現フェーズと最優先事項（重要）
- 現在は **フェーズ0＝全掲載無料で母数づくり**。
- **課金・決済まわりの実装はまだ行わない**（Stripe連携＝費用・オーナー承認が必要）。
- GuamJobsの当面の最優先は変わらず **「実在企業の求人を集める」こと**。マネタイズ実装はその後。

## 1. 市場と競合の把握（軽リサーチ）

### グアム労働市場の規模感【事実】
- グアムの就業者数は約 **68,440人**（2025年12月・速報, Guam Dept. of Labor BLS）。うち民間は約 **51,900人**（2025年3月）。
- 直近四半期で **+750人**。ホテル+180、建設+440と、観光・建設が牽引。
- ホテル業の平均時給は約 **$12.26**、週平均労働 約36時間 → 低〜中賃金・高回転の労働市場。
- 出典: [nminewsservice](https://www.nminewsservice.com/guam-employment-report-december-2025-jobs-labor/) / [Guam BLS](https://bls.guam.gov/) / [Pacific Island Times](https://www.pacificislandtimes.com/post/guam-employment-and-earnings-on-the-rise-but-global-conflict-poses-threats-to-local-economic-gains)

### 競合【事実】
- **GuamJobsOnline.com** — 地元求人ボード。**1掲載=30日の都度課金**が基本、請求書払いは$20手数料、**12ヶ月のバルクパッケージ**あり（＝月額サブスクではなく掲載単位課金＋年間パック）。具体額はサイト非公開。
- **HireGuam.com** — 地元。求人検索・履歴書作成など求職者ツール中心。
- **Indeed / SimplyHired / ZipRecruiter** — グローバル大手もグアム求人を扱う（掲載無料枠＋有料ブースト型が一般的）。
- 出典: [GuamJobsOnline Employer FAQ](https://guamjobsonline.com/employers/faq.php) / [SimplyHired Guam](https://www.simplyhired.com/search?l=guam) / [HireGuam](https://www.hireguam.com/)

### ここからの示唆
- 【事実ベース】市場は小さく（民間5万人規模）、賃金は低め＝**1件あたりの採用単価が低い**。企業が求人1件に大金は払いにくい。
- 【推測】既存地元競合は「掲載単位課金」中心。**月額サブスクは差別化になりうる**が、初期は掲載企業がゼロに近いため「サブスクを売る」より「まず載せてもらう」が先。
- 【推測】狙い目は観光関連（ホテル・飲食・ウォータースポーツ）の**継続的に人が要る＝反復採用**の業種。ここは月額と相性が良い。

## 3. 具体提案（3案）

### 【案1】フリーミアム＋有料アドオン（推奨・初期向き）
- **概要**: 求人掲載は**基本無料**（承認制）。目立たせたい企業に有料アドオンを販売:
  - Featured（トップ/一覧上位固定）
  - 「急募 / Urgent」バッジ
  - 掲載期間延長・複数枠パック
- **価格イメージ**【推測】: Featured 1件 **$25–40 / 30日**、急募バッジ **$10–15**、5件パック **$100前後**。
- **対象顧客**: まず全企業（無料で入口を下げる）→ 反復採用のホテル・飲食に有料枠を提案。
- **メリット**: 掲載無料で**企業も求人数も一気に集めやすい**（＝サイトの中身が埋まる）。有料は「欲しい人だけ」なので抵抗が低い。
- **デメリット**: 無料が主で**初期売上は小さい**。有料転換率次第。
- **初期適性**: ◎ / **収益化の早さ**: 中 / **実装コスト**: 中（`featured`フラグは既存、`isSample`同様に有料フラグ＋決済導線の追加。決済はStripe/PayPal等の連携が必要）。

### 【案2】都度課金（競合同型・キャッシュを最速で作る）
- **概要**: 1掲載 = 30日で課金。競合GuamJobsOnlineと同じ土俵。初回や紹介は割引。
- **価格イメージ**【推測】: 1掲載 **$29–49 / 30日**、5件バルク割引。
- **対象顧客**: 単発採用の中小企業・小売など。
- **メリット**: **1件売れれば即収益**。分かりやすい。
- **デメリット**: 掲載企業がまだ少ない初期に「有料の壁」を最初から立てると**掲載数が集まらず、求職者も来ない**（鶏卵問題を悪化）。競合と同型で差別化が弱い。
- **初期適性**: ○（ただし無料期間との併用前提）/ **早さ**: 早 / **実装コスト**: 中（決済連携必須）。

### 【案3】雇用主 月額サブスク（現行想定・スケール期の主軸）
- **概要**: 定額で一定枠を掲載し放題。現行方針「3ヶ月無料→月額」。
- **価格イメージ**【推測】: Basic **$29/月**（3枠）/ Pro **$79/月**（無制限＋Featured込み）。
- **対象顧客**: **継続的に採用する**ホテル・大型飲食・ツアー会社。
- **メリット**: **収益が安定・予測可能（MRR）**。反復採用の企業には割安で刺さる。
- **デメリット**: サブスクは「継続的な採用ニーズ＋十分な求職者流入」が前提。**掲載も応募も少ない初期には早すぎる**（価値を実感させられない＝解約）。
- **初期適性**: △ / **早さ**: 中 / **実装コスト**: 中〜高（サブスク課金・請求管理・枠制御ロジック）。

## 4. 推奨と段階的移行プラン

### 推奨（初期）: **案1 フリーミアム＋アドオン** を軸に、**案2の都度課金アドオン**を併走
理由（【推測】）:
- 初期最大の敵は「空っぽのサイト」。**無料掲載で企業と求人を早く増やす**ことが、求職者流入→サイト価値→将来の課金余地、の順で効く。
- 低賃金・小規模市場では高額サブスクは重い。**「無料で載る→効果を見て少額で目立たせる」**という段階が最も摩擦が少ない。
- 競合が都度課金/年間パック中心なので、**「基本無料＋アドオン」は明確な差別化**になる。

### 段階ロードマップ
1. **フェーズ0（今〜掲載10〜20社）**: 全掲載**無料**（現行の3ヶ月無料の精神）。目標は掲載数・応募数・求職者登録を積む。課金導線はまだ出さない。
2. **フェーズ1（企業が集まり応募が回り始めたら）**: **アドオン課金を導入**（Featured/急募/延長）。ここで初収益。同時に単発企業向けに**都度課金**も選べるように。
3. **フェーズ2（反復採用の常連が増えたら）**: **月額サブスク**を上位プランとして追加（アドオン込みで割安に）。常連をMRRへ移行。
4. **フェーズ3（サイパン→ミクロネシア展開時）**: 地域別プラン・地域スポンサー枠（案G/H）を追加。

> いずれも**決済連携（Stripe等）が必要**＝オーナーのアカウント・費用（決済手数料）が絡むため、導入時は必ず事前承認。コード実装は本提案の範囲外（承認後に別途）。

## 5. 秘書くん経由でオーナーに確認したいこと
1. **価格感の希望**: アドオン（Featured等）を **$25–40/件** 程度で想定してよいか。上限/下限の希望は?
2. **無料期間の考え方**: 現行「3ヶ月無料」は、(a)全社一律3ヶ月か、(b)掲載数が一定に達するまで全社無料（推奨）か、どちらの運用イメージか?
3. **決済手段**: 導入時、**Stripe / PayPal / 現地の請求書払い** のどれを優先したいか（グアム＝USドル圏、Stripe利用可）。決済手数料（例: Stripe 約2.9%+30¢/件）は許容範囲か?
4. **主ターゲット**: 最初に有料化を狙う業種は、想定通り **ホテル・飲食・ウォータースポーツ**（反復採用組）でよいか?
5. **競合価格の実額調査**: GuamJobsOnlineの実際の掲載料金はサイト非公開だった。**必要なら問い合わせ/実サイト確認で実額を取りにいく**が、やるべきか?
